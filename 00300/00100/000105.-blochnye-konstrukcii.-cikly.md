Суть цикла - многоразовое повторение кода внутри блока, кока выполняется условие. Единственным отличием от СанниБилдер в том, что количество условий может быть любое. Синтаксис написания следующий:

```
cs
int a = 0;

while (100 > a) {
 a++;
}
```
