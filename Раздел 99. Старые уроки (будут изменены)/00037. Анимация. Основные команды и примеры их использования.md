Мы научимся основным командам для проигрывания анимаций а так-же сделаем третью миссию с использованием анимацией.|wmysterio|wmysterio||||Если вы не читали урок о GTA Animation Manager, то советую прочитать <a href="/publ/gta_sa/dokumentacii_k_programmam/rukovodstvo_po_ispolzovaniju_programmy_gta_animation_maneger/54-1-0-6">руководство к этой программе</a>.

Для начала научимся загружать анимации. Сделать это можно командой:


```

04ED: load_animation "LOWRIDER"
```



<div class="panel panel-default">
 <div class="panel-body">
Здесь:<br>
"LOWRIDER" — название ifp-файла ( см. рисунок ниже )
 </div>
</div>

<!--IMG1--><img src="https://github.com/wmysterio/scm-scripting-lessons/raw/resources/_pu/0/15513934.png" /><!--IMG1-->

После этого нам обязательно нужно проверить: "загружена ли она?". Эта проверка осуществляется опкодом:


```

04EE: animation "LOWRIDER" loaded
```



После загрузки можно использовать анимации, что содержаться в ifp-файлах ( см. рисунок ниже ):

<!--IMG2--><img src="https://github.com/wmysterio/scm-scripting-lessons/raw/resources/_pu/0/77197739.png" /><!--IMG2-->

Теперь, что бы какой то актёр проиграл эту анимацию, нужно использовать опкод:


```

0605: actor 0@ perform_animation "M_SMKLEAN_LOOP" IFP "LOWRIDER" framedelta 4.0 loop 0 lockX 0 lockY 0 lockF 0 time 670
```



<div class="panel panel-default">
 <div class="panel-body">
Здесь:<br>
0@ — Имя переменной для актёра<br>
"prtial_gngtlkB" — название анимации<br>
"LOWRIDER" — название IFP-файла<br>
4.0 и 0 0 0 0 — оставляйте так, далее мы детально изучим эти параметры<br>
670 — время воспроизведения. Если поставить больше, то анимация будет повторяться. Длина любой анимации на рисунку выше выделено зелёным цветом
 </div>
</div>

Если мы загрузили анимацию, то естественно нам её нужно выгрузить с памяти когда она не нужна. Делается это опкодом:


```

04EF: release_animation "LOWRIDER"
```



Рассмотрим анимации, которые НЕ нужно загружать и НЕ нужно выгружать. Это анимации "PED.IFP", так как она загружается вместе с игрой и постоянно используется:

<!--IMG3--><img src="https://github.com/wmysterio/scm-scripting-lessons/raw/resources/_pu/0/84390702.png" /><!--IMG3-->

Рассмотрим другую анимацию - анимация телефона ( когда СЖ достаёт или прячет телефон ). Что бы актёр доставал телефон, используйте опкод:


```

0729: AS_actor $PLAYER_ACTOR hold_cellphone 1
```



<div class="panel panel-default">
 <div class="panel-body">
Здесь:<br>
$PLAYER_ACTOR - имя переменной актёра<br>
1 - достать телефон, 0 - спрятать
 </div>
</div>

После этого опкода обычно делают задержку в 2000 милисекунд, чтобы актёр окончательно взял телефон и приложил его к уху, или спрятал его в карман. 

<div class="alert alert-warning alert-dismissible" role="alert">
 <button type="button" class="close" data-dismiss="alert" aria-label="Close"><span aria-hidden="true">&times;</span></button>
**Примечание:** Сначала нужно загрузить модель телефона #CELLPHONE, иначе актёр приставит к уху только свою руку ;)
</div>

Так же актёра можно заставить говорить:


```

0967: actor 0@ move_mouth 4000 ms
```



<div class="panel panel-default">
 <div class="panel-body">
Здесь:<br>
0@ - хендл актёра<br>
4000 - время, сколько нужно шевелить губами
 </div>
</div>

И опкод, который заставляет прекратить актёра шевелить губами:


```

0968: actor 0@ stop_mouth
```



Есть ещё похожий опкод, который заставляет актёра шевелить губами, но это имеет вид больше жевания, нежели разговора:


```

05BA: AS_actor 2@ move_mouth 4000 ms
```



Так же актёров можно заставить сидеть. Делается это опкодом:


```

06B0: AS_actor 41@ sit_down 4000 ms
```



<div class="panel panel-default">
 <div class="panel-body">
Здесь:<br>
41@ - имя переменной для актёра<br>
4000 - указывается время, то есть сколько времени должен сидеть актёр
 </div>
</div>

Так же актёра можно заставить прыгать:


```

05BC: AS_actor 1@ jump 1
```



<div class="panel panel-default">
 <div class="panel-body">
Здесь:<br>
1@ - имя переменной для актёра<br>
1 - стиль прыжка. Так и оставляйте единицу, так как разницы между стилями не заметил
 </div>
</div>

Что бы заставить актёра присесть, используйте опкод:


```

04EB: AS_actor 65@ crouch 1
```



<div class="panel panel-default">
 <div class="panel-body">
Здесь:<br>
65@ - имя переменной для актёра<br>
1 - заставляет актёра присесть, если поставить 0 - то заставляет актёра встать
 </div>
</div>

Существует ещё проверка на приседание актёра:


```

0597: actor $PLAYER_ACTOR crouching
```



Так же есть полезная проверка на выполнение актёром какой-то анимации:


```

0611: actor 2@ performing_animation "LRGIRL_IDLE_TO_L0"
```



<div class="panel panel-default">
 <div class="panel-body">
Здесь:<br>
2@ - хэндл актёра<br>
"LRGIRL_IDLE_TO_L0" - имя анимации ( не IFP )
 </div>
</div>
